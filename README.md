# Fleet Management System

A full-stack web app to manage vehicles, drivers, trips and maintenance for a fleet.

## Status
In development (MVP: authentication + users).

## Features
### Current
- User registration & login (JWT)
- Protected API routes

### Planned
- Vehicle CRUD
- Driver CRUD + assignment to vehicles
- Maintenance scheduling
- Reports/dashboard

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: Microsoft SQL Server (MSSQL)

## Getting Started (Local Development)

### Prerequisites
- Node.js (18+ recommended)
- SQL Server (local or Docker)
- Git
